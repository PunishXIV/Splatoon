> [!Caution]
> 
> Only Splatoon 3.9.1.24 or higher will work on Phase 3. Currently, it is Testing version.
> 
> **= How to enable plugin testing:**
> 1. Go to `/xlsettings`
> 2. Go to "Experimental" tab
> 3. Enable "Get plugins testing builds" checkbox
> 4. Save settings by pressing a :floppy_disk: button in bottom right corner of the window
> 5. Open plugin installer
> 6. Find a plugin for which you want to receive testing version
> 7. Right click that plugin and enable "Receive plugins testing versions" menu item.
> 8. Update the plugin
